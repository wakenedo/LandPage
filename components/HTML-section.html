HTML
HyperText Markup Language

HTML (HyperText Markup Language) é a linguagem padrão para criar páginas na web. Foi criada por Tim Berners-Lee no final da década de 1980 e evoluiu ao longo das décadas até as especificações modernas do HTML5.

Breve história

Tim Berners-Lee desenvolveu o primeiro rascunho de HTML para permitir que documentos fossem conectados através de links (hyperlinks). Desde então, o HTML passou por versões (HTML 2.0, 3.2, 4.01) até o atual HTML5, que trouxe suporte semântico, multimídia e APIs modernas.

Exemplos de código

<!DOCTYPE html>
<html lang="pt-BR">
  <head>
    <meta charset="utf-8" />
    <title>Exemplo</title>
  </head>
  <body>
    <h1>Olá, HTML!</h1>
    <p>Este é um parágrafo.</p>
  </body>
</html>
