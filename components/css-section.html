CSS
Cascading Style Sheets
O CSS (Cascading Style Sheets) é a linguagem usada para descrever a apresentação das páginas web. Definindo o design, cores, espaçamentos, fontes, posicionamento e até animações.

Breve história

O CSS foi proposto em 1996 por Håkon Wium Lie, quando trabalhava com Tim Berners-Lee no CERN. A ideia era separar a camada de conteúdo (HTML) da camada de apresentação (CSS), tornando o código mais limpo, reutilizável e flexível. Desde então, passou por versões que evoluíram até o CSS3 e além.

Principais versões

  • CSS1 (1996)

    Estilos básicos como cores e fontes.

  • CSS2 (1998)

    Posicionamento de elementos e suporte à impressão.

  • CSS2.1 (2004)

    Revisão e compatibilidade entre navegadores.

  • CSS3 (2009+)

    Modularizado, trouxe transições, animações, gradientes, sombras e muito mais.

  • CSS (Atual)

    Evolução contínua com Flexbox, Grid Layout e variáveis.
