JavaScript

JavaScript é uma linguagem de programação usada para criar interatividade e funcionalidades dinâmicas em páginas web.

Breve história

JavaScript foi criado por Brendan Eich em 1995 enquanto trabalhava na Netscape, inicialmente para permitir interatividade em páginas web estáticas. Originalmente chamado de "Mocha" e depois "LiveScript", o nome foi alterado para "JavaScript" como parte de uma estratégia de marketing, aproveitando a popularidade do Java. Com o tempo, a linguagem evoluiu de um simples recurso de validação de formulários para uma ferramenta poderosa que hoje domina o desenvolvimento web, sendo usada tanto no lado do cliente quanto no servidor, com vasto suporte e um ecossistema em constante crescimento.

Exemplos de código

<!DOCTYPE html>
<html>
  <body>
    <button onclick="document.getElementById('demo').innerHTML = getRndInteger(0,20)">D20</button>
    <p id="demo"></p>
    <script>
      function getRndInteger(min, max) {
        return Math.floor(Math.random() * (max - min)) + min;
      }
    </script>
  </body>
</html>
